# Arena Legends Online

Arena Legends Online é um jogo web em desenvolvimento criado como projeto de aprendizado em programação e desenvolvimento de jogos online.

O projeto está sendo desenvolvido enquanto aprendo programação na prática, utilizando uma abordagem mais experimental e "vibe coding", focando tanto em entender o funcionamento dos sistemas quanto em construir algo jogável ao longo do processo.

## Estado do Projeto

O jogo ainda está em desenvolvimento e várias funcionalidades podem mudar, ser removidas ou refeitas conforme o projeto evolui.

Atualmente este projeto serve principalmente para:
- estudo
- prática de programação
- testes de sistemas
- experimentação de ideias

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Firebase

## Funcionalidades Atuais

- Sistema de autenticação/login
- Interface de jogo
- Estrutura inicial de personagens
- Sistemas básicos de batalha
- Integração com Firebase
- Salvamento de dados do jogador

## Objetivos Futuros

Algumas ideias planejadas para o futuro:

- sistema de progressão mais completo
- balanceamento de personagens
- melhorias visuais
- sistema multiplayer mais avançado
- ranking
- eventos
- novos modos de jogo

Este projeto ainda não está pronto para produção.
